## **主要内容**

- AiEDA（智能EDA框架）
  - 功能：
    - 支持芯片从RTL-GDS全流程设计，兼容商业EDA工具和开源EDA工具
    - 支持芯片设计过程各阶段特征提取，深入数据分析理解工具行为
    - 支持神经网络模型训练、测试和集成到EDA工具或芯片设计流程
    - 输入RTL或Netlist等数据，生成标签数据集，用于模型训练和验证
